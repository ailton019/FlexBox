Alinhamento de Itens em Contêiner Flexível

Usando a propriedade Flexbox no CSS, você pode criar contêineres flexíveis que ajustam a posição e o espaçamento dos itens automaticamente, facilitando o layout responsivo. Com ela, é possível alinhar, distribuir e organizar elementos de forma prática, permitindo que cada item se ajuste conforme o espaço disponível no contêiner.

Stretch (Padrão)

Estica os itens para que ocupem todo o espaço disponível, ajustando automaticamente seu tamanho dentro do contêiner.
